# Arata, a new shell

Arata is a simple shell written in C that I made for learning purposes *only*. There're no guarantees.

## Features

- [ ] Program execution
- [ ] Command history
- [ ] Setting and getting environment variables
- [ ] Job control
- [ ] Output redirection
- [ ] Bash keybinds*
